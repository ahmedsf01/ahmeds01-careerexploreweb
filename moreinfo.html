Here are some information on becoming a Server Administrator.

  1. Definition of a Server Administrator: CIS professionals who specialize in creating new user accounts, recovering lost passwords while keeping data secure, developing and implementing network maintenance standard operating procedures, diagnosing and repairing hardware and network malfunctions, ensuring securing firewalls are updated often, and providing management with regular reports.


  2. Education of a Server Administrator: One must have an IT bachelor's degree in which you must be trained in scripting, networks, web development, and technical communication. A masters degree that specializes in networks would be beneficial.


  3. Starting Salary of a Server Administrator: The starting salary, according to Zip Recruiter (a very famous recruitment site that has amassed over 110 million users), is about $63,663/year or $31/hour.


  4. Median Salary of a Server Administrator:According to Indeed (another famous Job Listing site), $88,330 is the median average annual salary for Windows Server Administrator.


  5. Experience to get a job as a Server Administrator: This engineer recruitment site has information on different job roles to fulfill for different engineers-for-hire worldwide. It specifies that it requires specialization (masters degree) in at least one operating system (Windows, Mac, or Linux). The site also claims that CIS professionals who would like to be Server Administrators must be able to script in languages such as Perl, Python, or Bash. Different organization require different certifications such as "Server+", "MCSE" or "RHCE" is beneficial.
